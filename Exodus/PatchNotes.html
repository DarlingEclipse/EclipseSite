Exodus Patch Notes

Update 0.7.4.0

2/24/2025

Description

Randomizer Rework - Improved Data Handling

Features

  • Made multiple quality of life changes for the Database system, including automatically loading required files (if they can be found) and saving the tree view expansion state when edits are made.
  • Rewrote a lot of background processing for information from the databases used for the Randomizer.
  • Rewrote Randomizer logic to be trick-dependant instead of general difficulty dependant (to be implemented soon)
  • Created infrastructure that will allow for more Randomizer features that the old system couldn't, including level order randomization and progression balancing.

NOTE

This update does remove some features that the Randomizer currently has. The only options will be to place Datacons, use custom locations, and the autobuild.

Bugfixes

  • Added a warning when attempting to automatically build an ISO without the excludezip.txt file.
  • Multiple fixes for Database editing.

Known Issues

  • Mods are still broken, but this is not at the top of the priority list currently.

Update 0.7.3.8

12/9/2024

Description

Improvements for the auto-build system.

Features

  • Reduced the number of files that need to be copied when building an ISO, reducing build times by a couple of minutes.
  • Added the "Mods and Replacements" view. This should make building modded ISOs more convenient.

Known Issues

  • Mods are, somehow, broken again. Tested with "Skip Intro FMVs" and "Highjump Always".
